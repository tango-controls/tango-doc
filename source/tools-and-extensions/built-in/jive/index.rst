Jive
====

:audience:`administrators, developers, users`

.. rubric:: Welcome to Jive documentation!


Jive is a standalone JAVA application designed to browse and edit the static **TANGO database**. Jive has been written
using **Swing** and need a JVM higher than 1.6.0. You will find in this documents the way to manage and create devices,
properties and classes. Jive also offers advanced search/selection features.

Download the project `here <https://bintray.com/tango-controls/maven/Jive/_latestVersion>`_

Find this project on `GitHub <https://github.com/tango-controls/jive>`_
